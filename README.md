# E-Library Management System

## 📌 Overview
A Flask-based web application to manage library books, users, and transactions efficiently.

## 🛠️ Tech Stack
- Python
- Flask
- SQL (Database)
- HTML, CSS

## ⚙️ Features
- Add, update, delete books  
- User management  
- Issue and return system  
- Search functionality  

## ▶️ How to Run
pip install -r requirements.txt  
python app.py  

Then open: http://127.0.0.1:5000/
